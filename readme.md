## 仿羊了个羊 微信小程序

### 用COCOS-2D开发的 **仿羊了个羊** 微信小程序


生成以及覆盖算法借鉴（抄袭）了[使用Vue3实现【羊了个羊】的算法方面全面解析！](https://cloud.tencent.com/developer/article/2134724) 具体的实现原理可以参考此文章

### 如何使用

安装cocos dashboard，下载编辑器（我使用的编辑器版本是3.6.1）， 然后导入此项目即可，我使用的编辑器版本是3.6.1。
